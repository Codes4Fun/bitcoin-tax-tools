These tools are to help you do your bitcoin taxes.

They are not guaranteed to be without risk, or to give you the biggest discount. You should go to a tax professional if you need that, but even if you do that consider using these tools a supplement, not just to double check your taxes but also to double check these tools.

They currently only do FIFO.
They use bitstamp exchange rates where rates are missing. This can be customized if you download and build gendata and replace bitcoin.*.rate files.

1) Gather transactions:
For bitcoin wallets generate download links for transaction data (in json).
To get the most privacy create your own insight API based bitcoin node, so you can download them locally.

For virtual wallets we use a transaction csv format.

This tool converts coinbase csv files to a transaction csv file.

For other virtual wallets create CSV files with your favorite spreadsheet editor (calc or excel) or a text editor, formatted with each transaction like this (quotes around each column):
"timestamp", "BTC delta", "exchange rate", "transaction id", "source"

where:
timestamp - is POSIX time, seconds passed since January 1, 1970.
BTC delta - this is how much was gained or lost in BTC in the transaction.
exchange rate - this was the exchange rate at the time of exchange. If you sold 2 BTC for 400, this should be 200 (400/2). If you don't know the exchange rate it will be generated later.
transaction id - this is the bitcoin transaction id, knowing this allows wallet to wallet transactions to not be counted as a sale/buy.
source - this is just for debugging purposes, it lets you know where this transaction came from (a coinbase csv or a bitcoin address, etc).

2) Generate input and output csv data from transaction csv/json files
input.csv and output.csv contain transactions across all wallets.
input.csv contains all transactions for acquired BTC.
output.csv contains all transactions of spent BTC.

3) Generate TXF and compute gains for a year
taxes.csv alternative format of TXF.
taxes.txf a document that can be imported into tax software.
taxes.txt a document showing your years income, cost, and gains.
